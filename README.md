# 🎵 Sky Player

Адаптивный PWA музыкальный плеер.  
Работает в браузере и устанавливается на смартфон как приложение.

## Возможности

- 🌊 **Моя волна** — авторадио по жанру и настроению
- 🔍 **Поиск** по исполнителю и названию
- ❤️ **Избранное** — лайкайте треки, они сохраняются на устройстве
- 📱 **PWA** — установите на телефон как приложение
- 🖥️ **Адаптивный** — работает на телефоне, планшете и десктопе

## Деплой на GitHub Pages

### 1. Создайте репозиторий

Залейте все файлы в новый **публичный** репозиторий на GitHub.

### 2. Включите GitHub Pages

`Settings` → `Pages` → `Source` → выберите **GitHub Actions**

### 3. Запушьте в main

```bash
git add .
git commit -m "Initial commit"
git push origin main
```

GitHub Action автоматически задеплоит сайт.  
Адрес будет: `https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПО/`

## Локальный запуск

Нужен любой HTTP-сервер (не просто открыть файл — ES modules требуют сервер):

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

Откройте `http://localhost:8080`

## Структура проекта

```
├── index.html          # Разметка всего приложения
├── style.css           # Стили (адаптивные)
├── app.js              # Точка входа, логика экранов
├── manifest.json       # PWA манифест
├── modules/
│   ├── parser.js       # Парсинг через Cloudflare Worker
│   ├── player.js       # Аудио движок и очередь
│   ├── storage.js      # LocalStorage (лайки, плейлисты, история)
│   ├── wave.js         # Логика "Моя волна"
│   └── ui.js           # Рендеринг DOM
├── assets/
│   ├── icon-192.png    # Иконка PWA
│   └── icon-512.png    # Иконка PWA (большая)
└── .github/
    └── workflows/
        └── deploy.yml  # Автодеплой на GitHub Pages
```

## Установка на телефон (PWA)

**Android (Chrome):** Открыть сайт → меню ⋮ → «Добавить на главный экран»  
**iOS (Safari):** Открыть сайт → кнопка «Поделиться» → «На экран "Домой"»
